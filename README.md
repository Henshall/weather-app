## About Project Setup

1) Create .env file with basic configuration depending on the envirnment
2) Run "composer install" command on project root
2) Run "npm install" command on project root
3) Run "npm run dev" command on project root
4) Run "php artisan serve" command on Project root to run the project

 => http://localhost:8000 open on the browser
